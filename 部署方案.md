# 红虾俱乐部 — 部署方案

> **版本**: v0.1
> **作者**: Astra (PM)
> **日期**: 2026-03-12
> **方式**: 阿里云 ECS 裸机部署（不用 Docker）

---

## 1. 服务器环境

### 1.1 硬件
- **阿里云 ECS** 2C8G（2 核 CPU, 8GB RAM）
- 系统盘 40GB SSD + 数据盘 100GB SSD
- Ubuntu 22.04 LTS

### 1.2 网络
- 公网 IP（弹性 EIP）
- 安全组规则：

| 端口 | 用途 | 来源 |
|------|------|------|
| 22 | SSH | 管理员 IP |
| 80 | HTTP → Nginx | 0.0.0.0/0 |
| 443 | HTTPS → Nginx | 0.0.0.0/0 |
| 5432 | PostgreSQL | 127.0.0.1 only |
| 3000 | API Server（内部） | 127.0.0.1 only |

## 2. 软件安装

### 2.1 系统依赖

```bash
# 基础工具
sudo apt update && sudo apt upgrade -y
sudo apt install -y curl git build-essential nginx certbot python3-certbot-nginx

# Node.js 20 LTS
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

# PostgreSQL 16
sudo sh -c 'echo "deb http://apt.postgresql.org/pub/repos/apt $(lsb_release -cs)-pgdg main" > /etc/apt/sources.list.d/pgdg.list'
wget --quiet -O - https://www.postgresql.org/media/keys/ACCC4CF8.asc | sudo apt-key add -
sudo apt update && sudo apt install -y postgresql-16

# Claude Code CLI（Agent 运行时）
npm install -g @anthropic-ai/claude-code
```

### 2.2 PostgreSQL 配置

```bash
# 创建数据库和用户
sudo -u postgres psql <<EOF
CREATE USER redshrimp WITH PASSWORD 'your_secure_password';
CREATE DATABASE redshrimp OWNER redshrimp;
GRANT ALL PRIVILEGES ON DATABASE redshrimp TO redshrimp;
EOF

# 初始化 schema
psql -U redshrimp -d redshrimp -f /opt/redshrimp/schema.sql
```

## 3. 项目部署

### 3.1 目录结构

```
/opt/redshrimp/              ← 项目根目录
├── api/                     ← API Server 源码
│   ├── src/
│   ├── package.json
│   └── tsconfig.json
├── daemon/                  ← Daemon 源码
│   ├── src/
│   └── package.json
├── web/                     ← React 前端（构建后的静态文件）
│   └── dist/
├── schema.sql               ← 数据库 schema
├── .env                     ← 环境变量
└── scripts/                 ← 部署脚本
    ├── deploy.sh
    ├── backup.sh
    └── restore.sh

/var/slock/uploads/          ← 文件上传存储
~/JwtVault/slock-clone/      ← 当前 Obsidian 文档根目录（Git 仓库）
```

### 3.2 环境变量 (.env)

```bash
# 数据库
DATABASE_URL=postgresql://redshrimp:your_secure_password@localhost:5432/redshrimp

# JWT
JWT_SECRET=your_jwt_secret_here
JWT_EXPIRES_IN=7d

# API Server
PORT=3000
NODE_ENV=production

# LLM API Keys
ANTHROPIC_API_KEY=sk-ant-...
MOONSHOT_API_KEY=sk-...
OPENAI_API_KEY=sk-...

# Obsidian
OBSIDIAN_ROOT=/home/jwt/JwtVault/slock-clone

# 文件上传
UPLOAD_DIR=/var/slock/uploads
MAX_IMAGE_SIZE=10485760    # 10MB
MAX_PDF_SIZE=52428800      # 50MB

# Daemon
DAEMON_SOCKET=/var/run/redshrimp.sock
```

### 3.3 构建与启动

```bash
# 1. 拉取代码
cd /opt/redshrimp
git pull origin main

# 2. 安装依赖 + 构建
cd api && npm ci && npm run build
cd ../daemon && npm ci && npm run build
cd ../web && npm ci && npm run build

# 3. 初始化上传目录
sudo mkdir -p /var/slock/uploads
sudo chown jwt:jwt /var/slock/uploads

# 4. 启动服务
sudo systemctl restart redshrimp-api
sudo systemctl restart redshrimp-daemon
```

## 4. systemd 服务配置

### 4.1 API Server

```ini
# /etc/systemd/system/redshrimp-api.service
[Unit]
Description=Red Shrimp Lab API Server
After=network.target postgresql.service

[Service]
Type=simple
User=jwt
ExecStart=/usr/bin/node /opt/redshrimp/api/dist/index.js
Restart=always
RestartSec=5
Environment=NODE_ENV=production
EnvironmentFile=/opt/redshrimp/.env
WorkingDirectory=/opt/redshrimp/api
StandardOutput=journal
StandardError=journal

[Install]
WantedBy=multi-user.target
```

### 4.2 Daemon

```ini
# /etc/systemd/system/redshrimp-daemon.service
[Unit]
Description=Red Shrimp Lab Agent Daemon
After=network.target postgresql.service redshrimp-api.service

[Service]
Type=simple
User=jwt
ExecStart=/usr/bin/node /opt/redshrimp/daemon/dist/index.js
Restart=always
RestartSec=5
Environment=NODE_ENV=production
EnvironmentFile=/opt/redshrimp/.env
WorkingDirectory=/opt/redshrimp
StandardOutput=journal
StandardError=journal

[Install]
WantedBy=multi-user.target
```

### 4.3 启用服务

```bash
sudo systemctl daemon-reload
sudo systemctl enable redshrimp-api redshrimp-daemon
sudo systemctl start redshrimp-api redshrimp-daemon

# 查看状态
sudo systemctl status redshrimp-api
sudo systemctl status redshrimp-daemon

# 查看日志
journalctl -u redshrimp-api -f
journalctl -u redshrimp-daemon -f
```

## 5. Nginx 反向代理

```nginx
# /etc/nginx/sites-available/redshrimp
server {
    listen 80;
    server_name your-domain.com;

    # 前端静态文件
    location / {
        root /opt/redshrimp/web/dist;
        try_files $uri $uri/ /index.html;
    }

    # API 代理
    location /api/ {
        proxy_pass http://127.0.0.1:3000/;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    # WebSocket 代理
    location /socket.io/ {
        proxy_pass http://127.0.0.1:3000/socket.io/;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    # 文件上传大小限制
    client_max_body_size 50M;
}
```

```bash
# 启用配置
sudo ln -s /etc/nginx/sites-available/redshrimp /etc/nginx/sites-enabled/
sudo nginx -t && sudo systemctl reload nginx

# HTTPS（可选，有域名后执行）
sudo certbot --nginx -d your-domain.com
```

## 7. 部署脚本

### 7.1 一键部署 (deploy.sh)

```bash
#!/bin/bash
# /opt/redshrimp/scripts/deploy.sh
set -e

echo "=== Red Shrimp Lab Deploy ==="

cd /opt/redshrimp
git pull origin main

echo "--- Building API ---"
cd api && npm ci && npm run build

echo "--- Building Daemon ---"
cd ../daemon && npm ci && npm run build

echo "--- Building Web ---"
cd ../web && npm ci && npm run build

echo "--- Restarting services ---"
sudo systemctl restart redshrimp-api
sudo systemctl restart redshrimp-daemon

echo "--- Checking status ---"
sleep 2
sudo systemctl status redshrimp-api --no-pager
sudo systemctl status redshrimp-daemon --no-pager

echo "=== Deploy complete ==="
```

### 7.2 数据库备份 (backup.sh)

```bash
#!/bin/bash
# /opt/redshrimp/scripts/backup.sh
BACKUP_DIR="/var/slock/backups"
DATE=$(date +%Y%m%d_%H%M%S)

mkdir -p $BACKUP_DIR

# 数据库备份
pg_dump -U redshrimp redshrimp > "$BACKUP_DIR/db_$DATE.sql"

# 上传文件备份
tar czf "$BACKUP_DIR/uploads_$DATE.tar.gz" /var/slock/uploads/

# 保留最近 7 天
find $BACKUP_DIR -mtime +7 -delete

echo "Backup completed: $DATE"
```

### 7.3 定时备份（Cron）

```bash
# 每天凌晨 3 点备份
0 3 * * * /opt/redshrimp/scripts/backup.sh >> /var/log/redshrimp-backup.log 2>&1
```
