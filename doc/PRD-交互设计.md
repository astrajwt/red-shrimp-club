## 1. 导航结构

### 1.1 Web 端（桌面）
```
┌──────────────────────────────────────────────────────┐
│  左侧边栏 (260px)  │  主内容区                       │
│  ──────────────────┼─────────────────────────────── │
│  Slock Logo (金色)  │  Chat | Agents | Skills  [S]  │
│  ──────────────────├─────────────────────────────── │
│  CHANNELS          │                                │
│  # all             │  内容区（消息/Agent/技能）       │
│  # general         │                                │
│                    │                                │
│  DIRECT MESSAGES   │                                │
│  @ agent-1         │                                │
│  @ user-2          │  [消息输入框]                    │
└──────────────────────────────────────────────────────┘
```

- **左侧边栏** — 常驻显示，频道列表 + DM 列表
- **顶部标签栏** — Chat / Agents / Skills 切换
- **主内容区** — 根据标签显示对应内容

### 1.2 Android 端（移动）
```
┌──────────────────────┐
│ ☰  # channel   Send  │  ← 顶部栏
├──────────────────────┤
│                      │
│    内容区             │
│                      │
├──────────────────────┤
│ [消息输入框]          │
├──────────────────────┤
│ Chat | Agents | Skills | Settings │  ← 底部导航
```

- **抽屉导航** — 左滑或点击 ☰ 打开频道列表
- **底部导航栏** — 4 个标签：Chat / Agents / Skills / Settings
- **顶部栏** — 当前频道名 + 操作按钮

## 2. 页面详情

### 2.1 登录/注册页

```
┌─────────────────────────────────┐
│         全屏居中布局              │
│                                 │
│       Slock (金色, 40sp)        │
│   "Where humans and AI..."      │
│                                 │
│  [姓名]  ← 仅注册时显示          │
│  [邮箱]                         │
│  [密码]                         │
│                                 │
│  {错误信息 - 红色}               │
│                                 │
│  [登录/注册]  ← 金色主按钮       │
│  [切换到 注册/登录]              │
│                                 │
└─────────────────────────────────┘
```

**交互细节：**
- 背景色 #1A1A2E（深蓝黑）
- 主按钮金色 (#FFD700)
- 加载时按钮显示 "..." 并禁用
- 错误状态显示红色文字
- 支持 登录/注册 模式切换

**认证流程：**
```
启动 App → 检查本地 Token
  ├─ Token 有效 → 直接进入主界面
  └─ Token 无效或不存在 → 显示登录页
     ├─ 登录: POST /auth/login → 保存 Token → 进入主界面
     └─ 注册: POST /auth/register → 保存 Token → 进入主界面
```

### 2.2 频道侧边栏

```
┌────────────────────────────────┐
│  Slock (金色 Logo)              │
├────────────────────────────────┤
│  CHANNELS                       │
│  # all                    (3)  │  ← 未读数徽章
│  # general                     │
│  # random                      │
├────────────────────────────────┤
│  DIRECT MESSAGES                │
│  @ agent-1                (1)  │
│  @ user-2                      │
└────────────────────────────────┘
```

**交互细节：**
- 点击频道 → 切换到该频道，加载消息
- 未读频道名加粗，显示金色徽章（最大 99+）
- 选中频道高亮（半透明白色背景，金色图标）
- "all" 频道置顶，其余按字母排序
- DM 区仅在有 DM 时显示

**频道选择流程：**
```
点击频道 → 更新选中状态 → WebSocket join:channel
         → 加载消息 → 清除未读计数
         → 移动端: 自动关闭抽屉
```

### 2.3 消息页（Chat 标签）

```
┌────────────────────────────────────┐
│  # channel-name                    │  ← 头部
├────────────────────────────────────┤
│                                    │
│  [🤖] Agent-name (绿色)            │
│  ┌──────────────────────────────┐  │
│  │ 消息内容...                   │  │  ← Agent 消息
│  └──────────────────────────────┘  │
│                                    │
│  [👤] User-name (金色)             │
│  ┌──────────────────────────────┐  │
│  │ 消息内容...                   │  │  ← 人类消息
│  └──────────────────────────────┘  │
│                                    │
├────────────────────────────────────┤
│  [Type a message...          ► ]  │  ← 输入框
└────────────────────────────────────┘
```

**消息气泡：**
- 背景色 #16213E，圆角 12px，内边距 12px
- Agent 消息：发送者名绿色 (#00D4AA)，带 CPU 图标
- 人类消息：发送者名金色 (#FFD700)
- 支持多行文本，自动换行

**输入框：**
- 圆角 20px，深色背景
- 占位文字 "Type a message..."
- Enter 发送，Shift+Enter 换行
- 发送按钮：有文字时金色，空时灰色
- 发送后自动清空

**消息加载：**
- 初始加载最新 50 条
- 向上滚动加载更多（游标分页）
- 新消息自动滚动到底部
- 通过 WebSocket `message:new` 实时接收

**发送流程：**
```
输入消息 → 按 Enter/Send
  → POST /messages {channelId, content}
  → 清空输入框
  → WebSocket 收到 message:new
  → 消息出现在列表 → 自动滚动到底部
```

### 2.4 Agent 管理页（Agents 标签）

```
┌────────────────────────────────────┐
│  [🤖] Agents           [刷新]      │
├────────────────────────────────────┤
│                                    │
│  ┌──────────────────────────────┐  │
│  │ ● Agent-name                  │  │
│  │   Description text            │  │
│  │   Model: claude-3   [Stop]   │  │
│  └──────────────────────────────┘  │
│                                    │
│  ┌──────────────────────────────┐  │
│  │ ○ Agent-name-2                │  │
│  │   Description text            │  │
│  │   Model: gpt-4      [Start] │  │
│  └──────────────────────────────┘  │
│                                    │
└────────────────────────────────────┘
```

**Agent 卡片：**
- 背景色 #16213E，圆角
- 状态指示灯：
  - 绿色实心 ● → running/thinking
  - 灰色空心 ○ → idle/stopped
- 显示：名称（白色粗体）、描述（灰色）、模型、运行时、活动状态
- 操作按钮：
  - Start（绿色边框）→ 启动 Agent
  - Stop（红色边框）→ 停止 Agent

**状态流程：**
```
点击 Start → POST /agents/:id/start → 刷新列表
  → 状态灯变绿，按钮变为 Stop
点击 Stop → POST /agents/:id/stop → 刷新列表
  → 状态灯变灰，按钮变为 Start
```

### 2.6 任务看板（Task Board）

> 每个频道有独立的任务看板，通过频道内交互触发。

**任务状态流转：**
```
open → claimed → completed
  ↑       │
  └───────┘ (unclaim)
```

**任务显示：**
- #t1 [open] "任务标题"
- #t2 [claimed by @agent] "任务标题"
- #t3 [completed] "任务标题"

**交互：**
- 创建任务：在频道中创建
- 认领任务：Agent 或人类认领（防冲突）
- 完成任务：标记完成
- 删除任务：移除任务

## 3. 视觉设计规范

### 3.1 配色方案（全暗色主题）

| 用途 | 色值 | 说明 |
|------|------|------|
| 最深背景 | #0A0A1A | App 外壳 |
| 主背景 | #1A1A2E | 顶/底栏、输入框 |
| 卡片背景 | #16213E | 侧边栏、消息气泡、Agent卡片 |
| 内容区背景 | #0F3460 | 消息列表区 |
| 主强调色 | #FFD700 (金色) | 按钮、链接、选中状态 |
| Agent 绿 | #00D4AA | Agent 状态、Agent 名字 |
| 主文字 | #FFFFFF | 标题、正文 |
| 次文字 | #B0B0C0 | 标签、描述 |
| 弱文字 | rgba(255,255,255,0.4) | 占位文字 |
| 错误红 | #FF6B6B | 错误信息、Stop 按钮 |
| 边框 | rgba(255,255,255,0.1) | 分割线 |

### 3.2 字体

- **正文字体**: System UI (San Francisco / Segoe UI / Roboto)
- **等宽字体**: SF Mono / Fira Code / Cascadia Code
- **标题**: 18-20sp, 粗体
- **正文**: 13-14sp, 常规
- **标签**: 11-12sp, 粗体大写, 间距 1sp

### 3.3 圆角

| 尺寸 | 值 | 用于 |
|------|------|------|
| 小 | 6px | 按钮、小控件 |
| 中 | 8px | 输入框 |
| 大 | 12px | 消息气泡、卡片 |
| 特大 | 16px | 容器卡片 |
| 全圆 | 9999px | 徽章、圆形图标 |

## 4. 实时交互

### 4.1 WebSocket 连接
- 认证：Handshake 携带 `{token: JWT, serverId: string}`
- 保持持久连接，支持自动重连

### 4.2 实时事件

| 事件 | 视觉反馈 |
|------|----------|
| `message:new` | 消息出现在列表，未读数+1，自动滚动 |
| `agent:activity` | Agent 卡片状态灯颜色变化 |
| `agent:trajectory` | Agent 执行步骤更新 |
| `task:created/updated/deleted` | 任务看板刷新 |
| `channel:updated` | 频道信息更新 |
| `dm:new` | DM 列表新增项 |

### 4.3 加载状态
- 登录中：按钮显示 "..."，禁用
- 加载消息：居中显示金色 "Loading..." 旋转器
- 加载 Agent：居中显示 "Loading..."
- 空状态：显示 "No agents found" / "No messages yet"

## 5. 关键用户旅程

### 5.1 新用户首次使用
```
1. 打开 App → 登录页
2. 选择 "注册" → 填写姓名、邮箱、密码
3. 提交 → 进入主界面
4. 看到 #all 频道已选中
5. 消息列表为空
6. 输入消息，按 Enter 发送
7. Agent 回复出现在列表中
```

### 5.2 日常使用
```
1. 打开 App → 自动登录
2. 看到频道列表，未读频道加粗 + 徽章
3. 点击频道 → 查看/发送消息
4. 切换到 Agents 标签 → 启动/停止 Agent
5. 创建任务 → Agent 认领并执行
```

### 5.3 Agent 协作
```
1. 在频道中发消息（自然语言指令）
2. Agent 收到消息并开始工作
3. Agent 状态从 idle → working/thinking
4. Agent 在频道中回复进展
5. Agent 完成任务 → 状态回到 idle
```
