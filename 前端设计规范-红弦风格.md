## 1. 色彩系统

```typescript
// src/styles/theme.ts
export const colors = {
  // 背景层（由深到浅）
  bgDarkest:  '#0A0608',   // 最深背景，近黑带红调
  bgDark:     '#120B0E',   // 主背景
  bgMedium:   '#1C1116',   // 侧边栏、面板
  bgLight:    '#2A1820',   // 悬停、选中项背景
  bgInput:    '#160E12',   // 输入框背景

  // 主色调（红系）
  crimson:    '#8B1A2A',   // 深红，主色调
  red:        '#C41E3A',   // 亮红，强调色
  redGlow:    '#E83A5A',   // 发光红，active/focus
  redFaint:   '#3D1C24',   // 极淡红，边框/分隔线

  // 点缀色（金系）
  gold:       '#B8860B',   // 暗金，次级强调
  amber:      '#D4A017',   // 琥珀，hover 高亮
  amberFaint: '#4A3010',   // 极淡金，背景点缀

  // 文字
  textPrimary:  '#F0E8E8', // 主文字，暖白
  textSecondary:'#C4A8A8', // 次文字
  textDim:      '#9A8080', // 弱化文字
  textFaint:    '#5C4A4A', // 占位符

  // 边框
  border:       '#3D1C24', // 默认边框
  borderGlow:   '#6B2035', // 焦点/激活边框

  // 状态色（融入风格）
  statusOnline:  '#B8860B', // 在线（琥珀金，非俗气绿）
  statusIdle:    '#5C4040', // 空闲（暗红）
  statusWorking: '#C41E3A', // 工作中（红）

  // 功能色
  error:    '#FF4444',
  success:  '#5A8A5A',
  warning:  '#D4A017',
}
```

## 2. 字体系统

```css
/* 正文：中文衬线 + 西文衬线，营造文学感 */
font-family: 'Noto Serif SC', 'Georgia', serif;

/* 代码、时间戳、ID：等宽字体 */
font-family: 'JetBrains Mono', 'Fira Code', monospace;

/* 标签、按钮：无衬线，保持清晰 */
font-family: 'Noto Sans SC', 'Inter', sans-serif;
```

字号规范：
```
xs:   11px  — 时间戳、标签
sm:   12px  — 次要文字、元信息
base: 14px  — 正文消息
md:   15px  — 频道名、用户名
lg:   18px  — 页面标题
xl:   24px  — 大标题
```

## 5. 微交互与氛围效果

```css
/* 整体背景轻微噪点纹理（营造胶片质感） */
body::before {
  content: '';
  position: fixed;
  inset: 0;
  background-image: url("data:image/svg+xml,...");  /* 噪点SVG */
  opacity: 0.03;
  pointer-events: none;
  z-index: 9999;
}

/* 新消息进入动画 */
.message-enter {
  animation: slideIn 0.2s ease-out;
}
@keyframes slideIn {
  from { opacity: 0; transform: translateY(8px); }
  to   { opacity: 1; transform: translateY(0); }
}

/* 链接悬停：红色下划线 */
a:hover { color: var(--red); text-decoration: underline; }

/* 滚动条风格 */
::-webkit-scrollbar { width: 4px; }
::-webkit-scrollbar-track { background: var(--bgDark); }
::-webkit-scrollbar-thumb { background: var(--crimson); border-radius: 2px; }
```

## 6. Android 端适配

Android 端保持同一色彩系统，使用 Jetpack Compose Material3 的暗色主题：

```kotlin
val SlockColorScheme = darkColorScheme(
    primary = Color(0xFFC41E3A),       // red
    onPrimary = Color(0xFFF0E8E8),
    primaryContainer = Color(0xFF8B1A2A),  // crimson
    background = Color(0xFF120B0E),    // bgDark
    surface = Color(0xFF1C1116),       // bgMedium
    onSurface = Color(0xFFF0E8E8),     // textPrimary
    secondary = Color(0xFFB8860B),     // gold
    outline = Color(0xFF3D1C24),       // border
)
```
